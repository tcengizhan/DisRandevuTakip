# DisRandevuTakip
C# Windows Forms ve SQL Server kullanılarak geliştirilen diş kliniği randevu takip sistemi.
## Kullanılan Teknolojiler

- C#
- Windows Forms
- SQL Server / MSSQL
- ADO.NET
- Visual Studio

## Proje Özellikleri

- Randevu ekleme
- Randevu güncelleme
- Randevu silme
- Randevu arama
- Bugünkü randevuları listeleme
- Tedavi notu ekleme
- Tedavi notu güncelleme
- Tedavi notu silme

## Proje Hakkında
Bu proje, diş kliniği için temel randevu takip işlemlerini kolaylaştırmak amacıyla geliştirilmiştir. Kullanıcılar randevu kayıtlarını yönetebilir, bugünkü randevuları görüntüleyebilir ve randevulara bağlı tedavi notları oluşturabilir.
